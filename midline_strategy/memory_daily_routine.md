## 每日开工三步走（必读）

每天开始工作前，按顺序执行：

### 1️⃣ 读取 CLAUDE.md
- 查看当前约定的规则、配置、代码规范
- 检查是否有新增/修改的全局约定

### 2️⃣ 读取日志文件
- `trading_bot.log` → 管道运行状态、信号、报错
- 检查昨夜/今早是否有异常

### 3️⃣ 读取 memory 文件
- `memory_worklog.md` → 当前进度，上次做了什么
- `memory_backtest_v4.md` → 回测改动记录
- `memory_feature_pipeline.md` → 特征管道设计说明

### 完成后
- 检查虚拟盘持仓和当日盈亏
- 根据以上信息决定当天工作内容
